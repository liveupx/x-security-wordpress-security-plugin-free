Complete WordPress security solution - Login protection, firewall, brute force protection, and more.

== Description ==

X Security is a comprehensive WordPress security plugin that protects your site from hackers, brute force attacks, and malicious activity.

= Features =

* **Login Protection** - Prevent brute force attacks with login lockout
* **Honeypot Field** - Catch bots with invisible form fields
* **Firewall** - Block malicious requests and bad bots
* **IP Blocking** - Block suspicious IP addresses
* **IP Whitelisting** - Whitelist trusted IPs
* **Activity Log** - Monitor all security events
* **User Enumeration Protection** - Prevent user discovery
* **Password Strength** - Enforce strong passwords
* **XML-RPC Protection** - Disable XML-RPC if not needed
* **File Editor Protection** - Disable theme/plugin editor
* **.htaccess Protection** - Server-level security rules

== Installation ==

1. Upload the plugin to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu
3. Go to X Security > Settings to configure
